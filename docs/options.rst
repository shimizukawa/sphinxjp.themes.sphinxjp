HTML theme options
===================

:hatena_bookmark:
    Show hatena-bookmark bookmarked count and link.
    default is False.
    See also: http://b.hatena.ne.jp/help

:hatena_star:
    Show hatena-star stars.
    default is False.
    See also: http://s.hatena.ne.jp/

:hatena_star_token:
    If you set `hatena_star = True`, you need set this token.
    See also: http://s.hatena.ne.jp/
