=================================
sphinxjp theme sample for Sphinx
=================================

Quick Start
=============

install:

.. code-block:: bash

    $ easy_install sphinxjp.themes.sphinxjp


setup your ``conf.py`` with:

.. code-block:: python

    html_theme = 'sphinxjp'

and run:

.. code-block:: bash

    $ make html

then you will get this page's style HTML output.

.. note::

   If you use sphinx-1.1.3 or before, you need install sphinxjp.themecore
   to use this theme plugin.
